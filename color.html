这是一个内容卡片的示例，您可以在这里展示各种信息。

卡片具有半透明背景，能够与背景渐变产生层次感。

当前CSS变量值

:root {
  --primary-bg: #6a11cb;
  --secondary-bg: #2575fc;
  --card-bg: rgba(255, 255, 255, 0.1);
  --text-color: #ffffff;
  --accent-color: #ff9e01;
  --nav-bg: rgba(0, 0, 0, 0.2);
  --button-color: #ff9e01;
}

如何使用这些颜色值

  1. 调整上面的颜色选择器，直到获得满意的配色方案
  2. 复制右侧显示的CSS代码
  3. 将代码添加到您网站的CSS文件最前面
  4. 确保您的CSS中使用这些变量（如：background: var(--primary-bg)）
  5. 保存并刷新您的网站查看效果

提示：对于卡片背景颜色，您可能需要调整Alpha通道（透明度）值以获得最佳效果。
